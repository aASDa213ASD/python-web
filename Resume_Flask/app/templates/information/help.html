List of available commands:
help       -   Obviously you already know what that does.
clear      -   Clears the terminal screen.
system     -   Displays system information and time.
whois      -   Who is aASDa213ASD?
projects   -   View coding projects.
ls         -   Lists available folders.
cd         -   Changes current directory.
skills     -   Displays skills, you can specify one by id.
login      -   Authentication system.
exit       -   Destroys current session.
passwd     -   Credentials tool-kit.
cookies    -   Cookies tool-kit, use set/wipe arguments.
